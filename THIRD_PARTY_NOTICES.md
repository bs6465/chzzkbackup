# 서드파티 고지

## Chzzk-Rekoda

치지직 스트림 녹화와 관련된 다음 구현 방식은 Chzzk-Rekoda를 참고했습니다.

- 치지직 `live-detail` API 기반 생방송 상태 확인
- Streamlink용 치지직 플러그인 구조
- 쿠키 헤더 전달 방식
- Streamlink 및 ffmpeg 프로세스 관리 방식

원본 프로젝트 정보는 다음과 같습니다.

- 프로젝트: https://github.com/munsy0227/Chzzk-Rekoda
- 라이선스: MIT

원본 참고 디렉터리는 이 저장소에 포함하지 않습니다. 서비스 동작에 필요한 부분만 현재 앱 구조에 맞게 리팩터링해 포함했습니다.
